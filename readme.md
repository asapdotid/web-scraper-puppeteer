## Install

```
clone this repository
```

Set up environment (.env) in root directory 

path domain must from https://coursehunters.net/{url for tutorial}

```
NODE_PATH=src
PATH_DOMAIN=
PATH_FILE=files
PATH_VIDEO=video
PATH_IMG=images
PATH_PDF=pdf
```

Running your code
```
$npm run scrape
```
